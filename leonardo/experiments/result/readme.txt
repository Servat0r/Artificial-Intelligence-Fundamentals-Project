Directory name is 
    run-{type of distance}-{polygons dimention}-{num of polygons}
type of distance : is the type of distance to evaluate the difference between two images (mse, ssim, ...)

Inside: 
    ga-{POPULATION_SIZE}-{P_CROSSOVER}-{P_MUTATION}-{MAX_GENERATIONS}-{HALL_OF_FAME_SIZE}-{CROWDING_FACTOR}
Different folder correspond to different Hyperparameters settings of the GA
